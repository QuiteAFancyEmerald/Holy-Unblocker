How to Set a Custom Icon / Disguise Your Tab

Want to change the tab title and favicon to make it look like something else?
Follow these quick steps:

  1. Go to the website you want to use as a disguise (e.g. Google, Classroom, Wikipedia, etc.)
  2. Drag the button below to your bookmarks bar (or right-click → Add to bookmarks):
Find Icon URL (Drag Me!)

Click the bookmarklet on the disguise site → it will show you the correct favicon URL.
Copy that URL, go back to SCBypass → open Settings → Tab Cloak → paste it in the Icon URL field.
You can also change the tab title there.
